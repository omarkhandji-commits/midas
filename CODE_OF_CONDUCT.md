# Code of Conduct

This project adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)
as its code of conduct.

## Summary

We pledge to make participation a harassment-free experience for everyone,
regardless of age, body size, visible or invisible disability, ethnicity, sex
characteristics, gender identity and expression, level of experience,
education, socio-economic status, nationality, personal appearance, race,
caste, color, religion, or sexual identity and orientation.

## Scope

This applies in all project spaces — issues, pull requests, discussions,
public communications referring to the project — and when an individual is
officially representing the project in public spaces.

## Enforcement

Instances of unacceptable behaviour may be reported through GitHub Security
Advisories on this repository or to the maintainer's email on their GitHub
profile. All reports are reviewed and investigated promptly and fairly.
