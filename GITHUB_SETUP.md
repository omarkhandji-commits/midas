## Manual Setup

1. Open `https://github.com/omarkhandji-commits/midas`.
2. Click the gear icon beside the repository About box.
3. Paste the description above.
4. Add the topics above.
5. Set the website field only if a real project website exists.
6. Upload the social preview image in Settings -> General -> Social preview.
